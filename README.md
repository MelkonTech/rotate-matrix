# Matrix Rotate

Rotate Matrix coding solution

## Prerequisites

- node: `v16.19.0`

## Run

1. Install dependencies

   ```sh
   npm install
   ```

2. Build

   ```sh
   npm run build
   ```

3. Run

   ```sh
   node cli.js input.csv > output.csv
   ```

## To run the test

    ```sh
    npm test
    ```
